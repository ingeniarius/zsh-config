zsh-config
==========

zsh configuration files

Featured
--------
- colored prompt with git branch name
- auto-complete
- separate config files

Setup
-----
```sh
git clone git://github.com/ingeniarius/zsh-config.git ~/Library/zsh
```

```sh
ln -s ~/Library/zsh/zshrc ~/.zshrc
```
